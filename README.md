## How to Get Started

### Deployed site on Heroku

<https://bowers-portfolio.herokuapp.com/>

### Run locally

#### Run backend

```
$ git@github.com:christopherbowers/portfolio_builder.git
$ cd portfolio_builder
$ npm i
$ npm start
```

#### Run frontend

```
$ cd client
$ npm i
$ npm start
```
